coron44_devices_touch3
======================
百度云OS 4.4项目

天语Touch3 高配适配项目源码公开 基于CM11

使用方法

进入coron项目

cd devices/

git clone https://github.com/HeXiaoShuai/coron44_devices_touch3.git

mv ./coron44_devices_touch3 ./touch3

cd ..

. build/envsetup.sh

ctouch3

make fullota

本项目持续更新

更新方法

cd devices/touch3

git pull
